# Minimal Translator

A minimal Chrome extension that translates webpage content and displays each translation below the original text.

## Install

1. Download this repository and unzip it.
2. Open `chrome://extensions/` in Chrome.
3. Enable **Developer mode**.
4. Drag the extracted `minimal-translator` folder onto the page.

## Use

1. Open a webpage.
2. Click **Minimal Translator** in the Chrome toolbar.
3. Choose the source and target languages.
4. Click **Translate this page**.
5. Scroll down to translate more content.

Text selected for translation is sent to Google Translate.
